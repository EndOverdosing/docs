---
title: "GET /api/scrape"
api: "GET https://vyla-api.pages.dev/api/scrape"
description: "Fetch raw sources from a single provider by TMDB ID"
---

Runs a single provider's scraper and returns raw sources and subtitles. This is the **internal building block** used by `/api/stream/movie`, `/api/stream/tv`, `/api/download/movie`, and `/api/download/tv` — those endpoints call `/api/scrape` once per provider in parallel so each gets its own isolated CPU budget.

Call it directly if you only want results from a specific provider, or want to fan out requests yourself.

<Note>
  Sources from `/api/scrape` are raw — no deduplication, no `download_url`, no `ffmpeg_command`, no proxied `vlc_url`. Use `/api/stream/movie` or `/api/stream/tv` for fully enriched results.
</Note>

### Query Parameters

<ParamField query="id" type="string" required>
  TMDB movie or series ID.
</ParamField>

<ParamField query="provider" type="string" required>
  Provider key. Must be one of: `moviedownloader`, `vixsrc`, `vidsrc`, `uembed`, `vidrock`, `rgshows`, `vidzee`, `embed02`
</ParamField>

<ParamField query="type" type="string" default="movie">
  `movie` or `tv`
</ParamField>

<ParamField query="season" type="number" default="1">
  Season number — TV only.
</ParamField>

<ParamField query="episode" type="number" default="1">
  Episode number — TV only.
</ParamField>

### Example Request

<CodeGroup>
```bash Single provider — movie
curl "https://vyla-api.pages.dev/api/scrape?type=movie&id=550&provider=vixsrc"
```

```bash Single provider — TV episode
curl "https://vyla-api.pages.dev/api/scrape?type=tv&id=456&season=1&episode=1&provider=vidrock"
```

```js Fan out across all providers
const providers = [
  "moviedownloader", "vixsrc", "vidsrc", "uembed",
  "vidrock", "rgshows", "vidzee", "embed02"
];
const base = "https://vyla-api.pages.dev/api/scrape?type=movie&id=550";

const results = await Promise.all(
  providers.map(p => fetch(`${base}&provider=${p}`).then(r => r.json()))
);

const allSources = results.flatMap(r => r.sources ?? []);
```

```python Python
import requests
from concurrent.futures import ThreadPoolExecutor

providers = ["moviedownloader","vixsrc","vidsrc","uembed","vidrock","rgshows","vidzee","embed02"]
base_params = {"type": "movie", "id": 550}

def scrape(p):
    return requests.get(
      "https://vyla-api.pages.dev/api/scrape",
      params={**base_params, "provider": p}
    ).json()

with ThreadPoolExecutor() as ex:
    results = list(ex.map(scrape, providers))

all_sources = [s for r in results for s in r.get("sources", [])]
```
</CodeGroup>

<Note>
  Firing all 8 provider requests with `Promise.all` and merging the results is exactly what `/api/stream/movie` and `/api/stream/tv` do internally — use those endpoints to get the enriched, deduplicated output without managing it yourself.
</Note>

### Valid Provider Keys

| Key | Provider |
|-----|---------|
| `moviedownloader` | MovieDownloader |
| `vixsrc` | VixSrc |
| `vidsrc` | VidSrc |
| `uembed` | UEmbed |
| `vidrock` | VidRock |
| `rgshows` | RgShows |
| `vidzee` | VidZee |
| `embed02` | Embed02 |

### Response

<ResponseField name="success" type="boolean" required>
  `true` if scraping succeeded.
</ResponseField>

<ResponseField name="sources" type="RawSource[]" required>
  Raw array of sources from the specified provider. No deduplication or enrichment applied.

  <Expandable title="RawSource object">
    <ResponseField name="url" type="string">
      Direct stream URL.
    </ResponseField>
    <ResponseField name="type" type="string">
      `hls`, `mp4`, or `mkv`
    </ResponseField>
    <ResponseField name="quality" type="string">
      e.g. `1080p`, `720p`, `unknown`
    </ResponseField>
    <ResponseField name="provider" type="string">
      Provider name.
    </ResponseField>
    <ResponseField name="audioTracks" type="AudioTrack[]">
      Available audio tracks with `language` and `label`.
    </ResponseField>
    <ResponseField name="headers" type="object">
      Required `Referer` and `Origin` headers for this CDN.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="subtitles" type="Subtitle[]" required>
  Raw subtitle array from the specified provider.

  <Expandable title="Subtitle object">
    <ResponseField name="url" type="string">
      Direct URL to the subtitle file.
    </ResponseField>
    <ResponseField name="label" type="string">
      Display name.
    </ResponseField>
    <ResponseField name="format" type="string">
      `vtt` or `srt`
    </ResponseField>
  </Expandable>
</ResponseField>
