## Endpoint

```
GET /api/scrape
```

Runs a single provider's scraper and returns raw sources and subtitles. This is the **internal building block** used by `/api/stream/movie`, `/api/stream/tv`, `/api/download/movie`, and `/api/download/tv` â€” those endpoints call `/api/scrape` once per provider in parallel so each gets its own isolated CPU budget.

Call it directly if you only want results from a specific provider, or want to fan out requests yourself.

<Note>
  Sources from `/api/scrape` are raw â€” no deduplication, no `download_url`, no `ffmpeg_command`, no proxied `vlc_url`. Use `/api/stream/movie` or `/api/stream/tv` for fully enriched results.
</Note>

---

## Parameters

<ParamField query="id" type="string" required>
  TMDB movie or series ID.
</ParamField>

<ParamField query="provider" type="string" required>
  Provider key. Must be one of the valid values listed below.
</ParamField>

<ParamField query="type" type="string" default="movie">
  `movie` or `tv`
</ParamField>

<ParamField query="season" type="string" default="1">
  Season number â€” TV only.
</ParamField>

<ParamField query="episode" type="string" default="1">
  Episode number â€” TV only.
</ParamField>

## Valid Providers

| Key | Provider |
|-----|----------|
| `moviedownloader` | 02MovieDownloader |
| `vixsrc` | VixSrc |
| `vidsrc` | VidSrc |
| `uembed` | Uembed |
| `vidrock` | VidRock |
| `rgshows` | RgShows |
| `vidzee` | VidZee |
| `embed02` | 02Embed |

## Example Requests

<CodeGroup>
```bash Single provider â€” movie
curl "https://vyla-api.pages.dev/api/scrape?type=movie&id=550&provider=vixsrc"
```

```bash Single provider â€” TV episode
curl "https://vyla-api.pages.dev/api/scrape?type=tv&id=456&season=1&episode=1&provider=vidrock"
```

```js Fan out across all providers (DIY)
const providers = ["moviedownloader","vixsrc","vidsrc","uembed","vidrock","rgshows","vidzee","embed02"];
const base = "https://vyla-api.pages.dev/api/scrape?type=movie&id=550";

const results = await Promise.all(
  providers.map(p => fetch(`${base}&provider=${p}`).then(r => r.json()))
);

const allSources = results.flatMap(r => r.sources ?? []);
```
</CodeGroup>

<Note>
  Firing all 8 provider requests with `Promise.all` and merging the results is exactly what `/api/stream/movie` and `/api/stream/tv` do internally â€” use those endpoints to get the enriched, deduplicated output without managing it yourself.
</Note>
