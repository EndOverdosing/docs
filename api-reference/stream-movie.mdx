## Endpoint

```
GET /api/stream/movie
```

Identical to [`/api/movie`](/api-reference/movie) — scrapes all providers for a movie and returns sources and subtitles.

## Parameters

| Name | Type   | Required | Description   |
|------|--------|----------|---------------|
| `id` | string | ✅       | TMDB movie ID |

## Example Request

```bash
curl https://vyla-api.pages.dev/api/stream/movie?id=550
```

## Example Response

```json
{
  "success": true,
  "results_found": 8,
  "sources": [
    {
      "url": "https://vixsrc.to/playlist/170060?token=...&expires=...&h=1",
      "type": "hls",
      "quality": "1080p",
      "provider": "VixSrc",
      "audioTracks": [{ "language": "eng", "label": "English" }],
      "headers": {
        "Referer": "https://vixsrc.to",
        "Origin": "https://vixsrc.to"
      }
    }
  ],
  "subtitles": [
    {
      "url": "https://vixsrc.to/playlist/170060?type=subtitle&rendition=eng&token=...&expires=...&edge=...",
      "label": "English",
      "format": "vtt"
    }
  ]
}
```

<Note>
  This is a path alias. For most use cases, prefer [`/api/movie`](/api-reference/movie).
</Note>
