---
title: "GET /api/test/{id}"
api: "GET https://vyla-api.pages.dev/api/test/{id}"
description: "Test individual sources for movies and TV episodes"
---

Test individual streaming sources to verify their availability and get direct stream URLs. This endpoint is useful for debugging or when you need to test a specific provider.

### Path Parameters

<ParamField path="id" type="string" required>
  TMDB ID. For movies use the movie ID, for TV use the series ID.
</ParamField>

### Query Parameters

<ParamField query="source" type="string" required>
  Source provider to test. Available sources:
  - `vixsrc`
  - `icefy` 
  - `vidzee`
  - `vidnest`
  - `vidsrc`
  - `vidrock`
  - `videasy`
</ParamField>

<ParamField query="season" type="number" required="false">
  Season number (required for TV episodes).
</ParamField>

<ParamField query="episode" type="number" required="false">
  Episode number (required for TV episodes).
</ParamField>

### Example Request

<CodeGroup>
```bash cURL
# Test movie source
curl "https://vyla-api.pages.dev/api/test/550?source=vixsrc"

# Test TV episode source  
curl "https://vyla-api.pages.dev/api/test/1396?season=1&episode=1&source=vixsrc"
```

```js JavaScript
// Test movie source
const res = await fetch("https://vyla-api.pages.dev/api/test/550?source=vixsrc");
const data = await res.json();

console.log(data.ok); // whether source is working
console.log(data.url); // proxied stream URL

// Test TV episode source
const tvRes = await fetch(
  "https://vyla-api.pages.dev/api/test/1396?season=1&episode=1&source=vixsrc"
);
const tvData = await tvRes.json();
```

```python Python

# Test TV episode source
```


</CodeGroup>

### Response

<ResponseField name="source" type="string" required>
  The source provider that was tested
</ResponseField>

<ResponseField name="id" type="string" required>
  TMDB ID that was tested
</ResponseField>

<ResponseField name="s" type="number | null" required>
  Season number (null for movies)
</ResponseField>

<ResponseField name="e" type="number | null" required>
  Episode number (null for movies)
</ResponseField>

<ResponseField name="ok" type="boolean" required>
  Whether the source is working and returned a valid stream
</ResponseField>

<ResponseField name="url" type="string | null">
  Proxied stream URL if successful, null if failed
</ResponseField>

<ResponseField name="raw_url" type="string | null">
  Original unproxied stream URL if successful, null if failed
</ResponseField>

<ResponseField name="elapsed_ms" type="number" required>
  Time taken to test the source in milliseconds
</ResponseField>

<ResponseField name="error" type="string | null" required>
  Error message if the test failed, null if successful
</ResponseField>

<Note>
  This endpoint tests individual sources in isolation. Use the `/api/movie` or `/api/tv` endpoints to test all sources simultaneously and get only working results.
</Note>

## Available Sources

- **vixsrc**: Reliable source with good quality streams
- **icefy**: Alternative source with moderate availability  
- **vidzee**: Good quality source when available
- **vidnest**: Fast response times, moderate availability
- **vidsrc**: Legacy source, often unavailable
- **vidrock**: Alternative source, hit or miss availability
- **videasy**: Decent quality source with reasonable response times
