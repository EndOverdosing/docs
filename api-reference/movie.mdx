---
title: Movie
description: Scrape streaming sources for a movie by TMDB ID
---

## Endpoint

```
GET /api/movie
```

## Parameters

| Name | Type   | Required | Description   |
|------|--------|----------|---------------|
| `id` | string | ✅       | TMDB movie ID |

## Example Request

```bash
curl https://vyla-api.pages.dev/api/movie?id=550
```

## Error Response

```json
{
  "success": false,
  "error": "Missing id"
}
```

<Note>
  Sources are sorted by quality (highest first) and filtered to English audio only. Always check `success` before using `sources`.
</Note>

<Note>
  TMDB movie IDs can be found at [themoviedb.org](https://www.themoviedb.org). The ID is in the URL — e.g. `themoviedb.org/movie/550` → ID is `550`.
</Note>

<Warning>
  The `headers` field on each source must be forwarded when fetching the stream directly. Pass them as base64 JSON to `/api/proxy` or `/api/download` via the `headers` param, or use `/api/download/movie` which handles this automatically.
</Warning>
