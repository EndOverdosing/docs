---
title: "GET /api/movie"
api: "GET https://vyla-api.pages.dev/api/movie"
description: "All working streaming sources and subtitle tracks for a movie, fetched from every provider in parallel"
---

Fetches all 6 providers concurrently for a movie, verifies each result with a live stream check, and returns only the sources that are actually working. Subtitle tracks are fetched and included automatically. Every stream URL in the response is already proxied and CORS-safe.

### Query Parameters

<ParamField query="id" type="string" required>
  TMDB movie ID. Find it at [themoviedb.org](https://www.themoviedb.org) — it's the number in the URL, e.g. `themoviedb.org/movie/550` → `550`.
</ParamField>

### Example Request

<CodeGroup>
```bash cURL
curl https://vyla-api.pages.dev/api/movie?id=550
```

```js JavaScript
const res = await fetch("https://vyla-api.pages.dev/api/movie?id=550");
const data = await res.json();

console.log(data.sources);   // verified, proxied stream sources
console.log(data.subtitles); // VTT subtitle tracks
```

```python Python
import requests

data = requests.get(
  "https://vyla-api.pages.dev/api/movie",
  params={"id": 550}
).json()
print(data["sources"])
print(data["subtitles"])
```

```ts TypeScript
const res = await fetch("https://vyla-api.pages.dev/api/movie?id=550");
const data = await res.json();
const sources = data.sources;
const subtitles = data.subtitles;
```
</CodeGroup>

### Response

<ResponseField name="sources" type="Source[]" required>
  Array of verified, proxied stream sources. Only working sources are included.

  <Expandable title="Source object">
    <ResponseField name="source" type="string">
      Internal provider key, e.g. `vixsrc`, `vidzee`.
    </ResponseField>

    <ResponseField name="label" type="string">
      Human-readable provider name, e.g. `vixsrc`, `VidZee`.
    </ResponseField>

    <ResponseField name="url" type="string">
      Proxied stream URL — routed through `/api` with CORS and full m3u8 segment rewriting applied. Pass directly to HLS.js or any player.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="subtitles" type="Subtitle[]" required>
  Array of available subtitle tracks for the movie. Empty array if none found.

  <Expandable title="Subtitle object">
    <ResponseField name="label" type="string">
      Language name of the subtitle track, e.g. `English`, `Spanish`.
    </ResponseField>

    <ResponseField name="file" type="string">
      Direct URL to the VTT subtitle file.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object | null">
  TMDB metadata for the movie. `null` if no `TMDB_API_KEY` is configured.
</ResponseField>
