---
title: "GET /api/tv"
api: "GET https://vyla-api.pages.dev/api/tv"
description: "All working streaming sources for a TV episode, fetched from every provider in parallel"
---

Fetches all 7 providers concurrently for a TV episode, verifies each result with a live stream check, and returns only the sources that are actually working. Every URL in the response is already proxied and CORS-safe.

### Query Parameters

<ParamField query="id" type="string" required>
  TMDB **series** ID — not episode ID. Find it at [themoviedb.org](https://www.themoviedb.org), e.g. `themoviedb.org/tv/1396` → `1396`.
</ParamField>

<ParamField query="season" type="number" required>
  Season number.
</ParamField>

<ParamField query="episode" type="number" required>
  Episode number.
</ParamField>

### Example Request

<CodeGroup>
```bash cURL
curl "https://vyla-api.pages.dev/api/tv?id=1396&season=1&episode=1"
```

```js JavaScript
const res = await fetch(
  "https://vyla-api.pages.dev/api/tv?id=1396&season=1&episode=1"
);
const data = await res.json();
console.log(data.sources);
```

```python Python
import requests

data = requests.get(
  "https://vyla-api.pages.dev/api/tv",
  params={"id": 1396, "season": 1, "episode": 1}
).json()
```

```ts TypeScript
const res = await fetch(
  "https://vyla-api.pages.dev/api/tv?id=1396&season=1&episode=1"
);
const data = await res.json();
const sources = data.sources;
```
</CodeGroup>

### Response

<ResponseField name="sources" type="Source[]" required>
  Array of verified, proxied stream sources. Only working sources are included.

  <Expandable title="Source object">
    <ResponseField name="source" type="string">
      Internal provider key, e.g. `vixsrc`, `vidnest`.
    </ResponseField>

    <ResponseField name="label" type="string">
      Human-readable provider name.
    </ResponseField>

    <ResponseField name="url" type="string">
      Proxied stream URL — routed through `/api` with CORS and full m3u8 segment rewriting applied. Pass directly to HLS.js or any player.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object | null">
  TMDB metadata for the episode. `null` if no `TMDB_API_KEY` is configured.
</ResponseField>

<Note>
  The `id` parameter refers to the **series**, not an individual episode. Season and episode are passed as separate parameters.
</Note>
