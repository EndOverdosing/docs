---
title: "GET /api/stream/tv"
api: "GET https://vyla-api.pages.dev/api/stream/tv"
description: "Enriched TV episode sources with download URLs, ffmpeg commands, and deduplication across all providers in parallel"
---

Scrapes all 8 providers for a TV episode **in parallel** — each in an isolated request with its own CPU budget — and returns fully enriched sources. Unlike [`/api/tv`](/api-reference/tv), this endpoint produces significantly more sources, deep-deduplicates them, and attaches pre-built `download_url`, `ffmpeg_command`, and proxied `vlc_url` on every source.

`/api/download/tv` is an alias — both endpoints return the identical response.

### Query Parameters

<ParamField query="id" type="string" required>
  TMDB **series** ID — not episode ID. Find it at [themoviedb.org](https://www.themoviedb.org), e.g. `themoviedb.org/tv/456` → `456`.
</ParamField>

<ParamField query="season" type="number" default="1">
  Season number.
</ParamField>

<ParamField query="episode" type="number" default="1">
  Episode number.
</ParamField>

### Example Request

<CodeGroup>
```bash cURL
curl "https://vyla-api.pages.dev/api/stream/tv?id=456&season=1&episode=1"
```

```js JavaScript
const res = await fetch(
  "https://vyla-api.pages.dev/api/stream/tv?id=456&season=1&episode=1"
);
const data = await res.json();

// Get best MP4 download
const mp4 = data.sources.find(s => !s.is_hls && s.download_url);
if (mp4) window.location.href = "https://vyla-api.pages.dev" + mp4.download_url;
```

```python Python
import requests

data = requests.get(
  "https://vyla-api.pages.dev/api/stream/tv",
  params={"id": 456, "season": 1, "episode": 1}
).json()

for s in data["sources"]:
    print(s["quality"], s["type"], s["provider"])
```

```ts TypeScript
const res = await fetch(
  "https://vyla-api.pages.dev/api/stream/tv?id=456&season=1&episode=1"
);
const data: StreamTVResponse = await res.json();
```
</CodeGroup>

### Response

<ResponseField name="success" type="boolean" required>
  `true` if scraping succeeded.
</ResponseField>

<ResponseField name="tmdb_id" type="string" required>
  The TMDB series ID that was queried.
</ResponseField>

<ResponseField name="season" type="string" required>
  The season number that was queried.
</ResponseField>

<ResponseField name="episode" type="string" required>
  The episode number that was queried.
</ResponseField>

<ResponseField name="results_found" type="number" required>
  Total number of deduplicated sources returned.
</ResponseField>

<ResponseField name="sources" type="EnrichedSource[]" required>
  Fully enriched, deduplicated stream sources sorted by quality.

  <Expandable title="EnrichedSource object">
    <ResponseField name="provider" type="string">
      Which provider this source came from.
    </ResponseField>

    <ResponseField name="quality" type="string">
      e.g. `1080p`, `720p`, `unknown`
    </ResponseField>

    <ResponseField name="type" type="string">
      `hls`, `mp4`, or `mkv`
    </ResponseField>

    <ResponseField name="url" type="string">
      The real, unwrapped stream URL after proxy chain resolution.
    </ResponseField>

    <ResponseField name="is_hls" type="boolean">
      `true` if this is an HLS/m3u8 stream.
    </ResponseField>

    <ResponseField name="referer" type="string | null">
      The `Referer` header required by the CDN.
    </ResponseField>

    <ResponseField name="ffmpeg_command" type="string">
      Ready-to-run ffmpeg command with all required headers embedded.
    </ResponseField>

    <ResponseField name="download_url" type="string | null">
      Relative URL for direct browser download. `null` for HLS sources — use `ffmpeg_command` instead.
    </ResponseField>

    <ResponseField name="vlc_url" type="string">
      Pre-built proxied URL. Open in VLC or any URL-capable player directly.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="subtitles" type="Subtitle[]" required>
  Deduplicated subtitle array.

  <Expandable title="Subtitle object">
    <ResponseField name="url" type="string">
      Direct URL to the subtitle file.
    </ResponseField>
    <ResponseField name="label" type="string">
      Display name, e.g. `English`, `English [CC]`
    </ResponseField>
    <ResponseField name="format" type="string">
      `vtt` or `srt`
    </ResponseField>
  </Expandable>
</ResponseField>

<Note>
  Duplicate sources (same underlying file wrapped by different provider proxy chains) are automatically removed.
</Note>

<Warning>
  HLS/playlist URLs cannot be downloaded as a single file via a browser link. They must be muxed to MP4 using the provided `ffmpeg_command`.
</Warning>
